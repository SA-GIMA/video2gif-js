# 视频转 GIF 工具 - Node.js 版

**开发者：商建航**

## 简介

一个本地运行的视频转 GIF 工具，通过浏览器操作，支持参数调节和批量转换。本项目为原 Python 版本的 Node.js 架构重写版，拥有与之完全一致的 UI 界面和并行转换逻辑。

## 环境要求

- **Node.js**: v14 或更高版本。
- **ffmpeg**: 必须在系统的环境变量 PATH 中，用于视频分析与转换。

## 快速启动

1. 在当前目录下打开终端，安装依赖：
   ```bash
   npm install
   ```
2. 启动服务：
   ```bash
   npm start
   ```
3. 在浏览器中打开 http://127.0.0.1:5050 即可使用。

## 使用步骤

1. **上传视频** — 将视频文件拖拽到页面上传区域，或点击选择文件。支持一次上传多个文件（最多 10 个）。
2. **调整参数** — 上传后会显示参数设置面板，可调节以下选项：
   - **帧率 (FPS)**：1-30，值越高动画越流畅，文件越大。推荐 10-15。
   - **宽度**：输出 GIF 的宽度（像素），高度按比例自动计算。推荐 320-480。
   - **起始时间**：从视频的第几秒开始截取，默认 0（从头开始）。
   - **持续时长**：截取多少秒，默认 0 表示截取到视频结尾。
   - **画质模式**：
     - 高质量：使用 ffmpeg 双 pass 调色板技术，色彩更准确，速度较慢。
     - 快速：单次处理，速度更快但色彩可能有色带。
   - **循环次数**：0 表示无限循环，1 表示只播放一次。
3. **开始转换** — 点击「开始转换」按钮，所有文件会并行处理（最多同时 3 个），页面实时显示进度。
4. **下载结果** — 转换完成后，每个文件旁会出现「下载」和「预览」按钮。

## 支持的视频格式

MP4、AVI、MOV、MKV、WebM、FLV、WMV

## 文件限制

- 单个文件最大 500MB。
- 每次服务启动时会自动清理上次残留的临时文件。也可以通过页面上的清理按钮手动清理服务器。
